# L'Échappatoire Royale

Livre dont vous êtes le héros sur le thème de Louis XVI et la Révolution française.

## Structure du projet

```
/
├── index.html              Page d'accueil (menu de démarrage)
├── README.md               Documentation
├── assets/
│   └── css/
│       ├── common.css      Styles partagés (variables, body, animations, responsive)
│       ├── index.css       Styles de la page d'accueil
│       └── game.css        Styles du jeu (chapitres, choix, fins, stats)
├── scripts/
│   ├── common.js           Étoiles, particules, bouton son
│   ├── chapters.js         Données des 40+ chapitres
│   └── game.js             Logique du jeu (affichage, choix, stats)
└── pages/
    └── game.html           Page du jeu
```

## Lancer le site

Ouvrir `index.html` dans un navigateur web. Aucun serveur requis.

## Technologies

- HTML5, CSS3, JavaScript vanilla
- [Font Awesome 6.4.0](https://fontawesome.com/) pour les icônes
- [Google Fonts](https://fonts.google.com/) : Cinzel (titres) et Crimson Text (texte)
